**Corona Discharge Location Detection in High Voltage Transmission Lines**  
**Description:**  
This project focuses on utilizing image processing techniques to detect the location of corona discharge in high voltage transmission lines. By leveraging image processing algorithms, this tool aims to pinpoint the areas of corona discharge, assisting in the maintenance and monitoring of high voltage infrastructure.  
**Features:**
* Image processing to identify corona discharge hotspots
* Visualization of corona discharge locations on transmission line images
